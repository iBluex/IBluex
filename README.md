## ЁЯУб How I Analyze a Protocol

```
1. Read the IDL                  тЖТ what functions exist, what accounts they touch
2. Trace recent transactions     тЖТ who's using it and how
3. Check authority accounts      тЖТ is it actually decentralized?
4. Understand token mechanics    тЖТ inflation, vesting, fee capture
5. Look at the team's past code  тЖТ reputation is on-chain too
6. Apply market context          тЖТ 4 years of pattern recognition kicks in

Only then I have an opinion.
```
